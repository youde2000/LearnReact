# 处理Token失效
用户长时间未在网站中做任何操作并且在规定的失效时间到达之后，当前的Token就会失效
> 前端如何知道Token已经失效了？
+ 通常在Token失效之后再去请求后端接口，后端会返回401状态码，前端可以监控这个状态做后续的操作
> Token失效了前端需要做什么？
+ 在axios拦截中监控401状态码
+ 清除失效token，跳转到登录页
